Requirements :
NODE.JS v14.17.0 and NPM 6.14.13

How to run server -

in client directory:

```
npm install
```

```
npm run build
```

in server directory:

```
npm install
```

```
npm start
```

Hosts on port 5000;

For Developers :
when building frontend, make sure to change TYPE:"DEVELOPMENT" to TYPE:"PRODUCTION" in client/src/build-type.json
